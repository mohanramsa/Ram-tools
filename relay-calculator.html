📋 Relay Testing Guide for Electrical Engineers

Protective relays are critical components in electrical power systems. They detect faults and protect equipment like transformers, motors, and transmission lines from damage. This section provides a basic understanding of relay testing methods used by electrical engineers in real-world field conditions.

1. Overcurrent Relay Testing

Overcurrent relays operate when current exceeds a preset value. They are commonly used in feeders and distribution systems.

  • Check pickup current setting
  • Inject current using relay test kit
  • Measure operating time
  • Verify inverse time characteristics

Example: If relay pickup is set at 100A, it should operate when injected current exceeds this value.

2. Differential Relay Testing

Differential protection is used for transformers and generators. It compares current entering and leaving the equipment.

  • Check CT polarity and ratio
  • Perform stability test (no trip condition)
  • Perform differential (trip) test
  • Verify percentage bias settings

If incoming and outgoing current differ beyond limit, relay will trip instantly.

3. Distance Protection Relay

Distance relays are used in transmission lines. They measure impedance (Z = V/I) to detect faults.

  • Zone 1: Instant trip (80–90% line)
  • Zone 2: Time delay backup
  • Zone 3: Remote backup protection

Testing involves injecting voltage and current to simulate fault conditions and verifying relay operation in each zone.

Why This is Important

Relay testing ensures system reliability and safety. Improper relay settings can cause equipment damage or unnecessary power outages. Electrical engineers must regularly test and verify relay performance in substations and industrial systems.

📐 NEF Relay — Key Facts • Uses residual current = I_R + I_Y + I_B (3I₀) via core-balance CT or residual 3-CT connection
• Pickup: 10–40% of rated CT primary to detect high-resistance earth faults
• NI / EI curves most common for feeder protection
• EF relay TMS must be lower than phase OC TMS for selective tripping
• Typical use: 11kV / 33kV feeders, cable circuits, transformer outgoing feeders
Restricted Earth Fault (REF) provides high-speed protection for earth faults within the protected zone (transformer winding, generator stator). Compares neutral CT current with residual of phase CTs.
Operate: Idiff = |Ineutral − Iresidual| ≥ Ipickup → Instantaneous trip
High-Z: stabilising resistor | Low-Z: numerical relay (GE P642, ABB RET670)
Typical: 5–20% of rated current
Must satisfy: Vk ≥ 2 × (RCT + RL) × I_fault_sec
📐 REF Relay — Key Facts • High-speed instantaneous — trip time < 40 ms
• Immune to external (through) faults — only detects faults inside zone
• High-Z REF: Stabilising resistor (Rs) prevents mal-operation on CT saturation. Vs = Iop × Rs
• Low-Z REF: Numerical relay digitally computes Idiff = I_neutral − (IR+IY+IB)/3
• Standards: IEC 60255 | IEEE C37.91 | BS 7671
• Applications: Power transformers (Dyn11), generator stator windings, reactors
Sensitive Earth Fault (SEF) detects very low-level earth fault currents (1–50A primary) on high-impedance or resistance-earthed systems. Used on 11kV / 33kV networks with Petersen coil or NGR earthing.
Operate: I0 ≥ ISEF_pickup → Definite time trip or alarm
Use CBCT (ring CT encircling all 3 phases)
Typically 1–50A on HRG / Petersen systems
Very sensitive: typical 0.05–0.5A secondary
Leave blank for Petersen / isolated system
SEF uses definite time delay (not IDMT)
📐 SEF Relay — Key Facts • Use where normal EF relay cannot detect fault currents < 5% of CT primary
• Requires core-balance CT (CBCT) — single CT encircles all 3 phases + neutral
• NGR earthed: I_fault = V_phase / NGR — limits fault current to safe level
• Petersen Coil: coil inductance tunes to system capacitance → near-zero fault current; only wattmetric component detectable
• Isolated: fault current = 3 × V × ωC₀ (capacitive) — directional SEF required
• Standards: IEC 60255-151 | IEEE C37.113 | NPAG

Step Calculation Formula / Note

2. LV Rated Current I_LV = kVA×1000 / (√3 × V_LV×1000) Full load current on LV winding

4. HV SC Current at Test Voltage Isc_HV = I_HV_rated × (V_test / Vz) Proportional: since V_test < Vz, Isc_HV < rated. Cross-check against measured value.
5. LV SC Current (Normal Tap) Isc_LV = Isc_HV × (V_HV / V_LV) Short circuit current reflected to LV side via voltage ratio
6. Stability Check Measured Isc vs Calculated Isc If measured ≈ calculated (within ±5%) → ✅ PASS. Deviation >5% → ⚠️ investigate
